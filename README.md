# # Inventory Optimization in E-commerce using the EOQ Model

## Project Overview
This project explores inventory optimization in e-commerce using the Economic Order Quantity (EOQ) model.

The goal is to derive the EOQ formula mathematically, implement it in Python, and validate it through simulation (without using real datasets).
